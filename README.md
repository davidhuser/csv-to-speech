# csv-to-speech

Uses Google TTS to speak words and converts the .mp3 files to .wav files with sample rate 8KHz, 16 bit, mono, Codec: PCM 16 LE (s16l)

Requirements:

* Python3
* sox, e.g. ```sudo apt install sox```

Installation:

```bash
pip install -r requirements.txt
```

Run:

```bash
python run.py
```
